Habit Tracking website

The Her Level Up Challenge website is a personal project that was created from scratch, including idea generation, design, coding, and content creation. The website offers a 21-day challenge for women to level up internally and externally by following 7 simple tasks to establish healthy habits. To develop the website, HTML, CSS, and JavaScript were utilized to make it responsive and visually appealing. The website is designed to make it easy to join the challenge, which was created on TikTok to inspire women to follow along and create a community of women who want to level up and hold each other accountable.

HTML | CSS | JavaScript | Bootstrap
